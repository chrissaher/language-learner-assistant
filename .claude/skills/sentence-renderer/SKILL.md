# sentence-renderer Skill for French Learning Assistant

## Overview
Converts JSON sentence files (generated by `sentence builder`) into styled, browseable HTML pages. Automatically scans `data/sentences/` for unrendered files and creates matching `.html` files in `data/sentences-html/` using the shared `styles.css`.

## Invocation
`render sentences`

---

## Workflow

### Step 1: Scan for Unrendered Files
- Read all `.json` files in `data/sentences/`
- For each JSON file, check if a corresponding `.html` file exists in `data/sentences-html/` with the same stem
- If `.html` already exists and is newer than `.json`, skip it (already rendered)
- If `.html` doesn't exist or is older than `.json`, mark for rendering

### Step 2: For Each Unrendered File
- Parse the JSON
- Extract metadata: `word`, `base_form`, `word_type`, `generated_at`, and sentences array
- Validate JSON structure (all required fields present)

### Step 3: Generate HTML
Create an HTML file with:
- **Header**: word name, word type, level (A1), generation timestamp
- **Sections**: grouped by level (simple / medium / complex)
- **Cards**: for each sentence, show target (French), translation (English), grammar note, with styling based on difficulty

### Step 4: Output
- Save to `data/sentences-html/[word_stem].html`
- Example: `data/sentences/manger.json` → `data/sentences-html/manger.html`

### Step 5: Summary
Display to user:
```
✅ Rendered X sentences file(s) to HTML
  - data/sentences-html/word1.html
  - data/sentences-html/word2.html

Open any file in your browser to view. All files use dark mode support via styles.css.
```

## HTML Template

```html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Sentences: {word} ({word_type}) — A1</title>
  <link rel="stylesheet" href="../../styles.css">
</head>
<body>
<div class="container">
  <header>
    <h1>{word}</h1>
    <p class="focus">{base_form} · {word_type} · A1 · French</p>
    <p class="meta">Generated {timestamp}</p>
  </header>

  <nav>
    <a href="../../index.html">← Back to Index</a>
  </nav>

  <section id="simple">
    <h2><span class="section-badge badge-sentences">Sentences</span>Simple (3)</h2>
    <!-- Sentence cards for level: "simple" -->
  </section>

  <section id="medium">
    <h2><span class="section-badge badge-sentences">Sentences</span>Medium (3)</h2>
    <!-- Sentence cards for level: "medium" -->
  </section>

  <section id="complex">
    <h2><span class="section-badge badge-sentences">Sentences</span>Complex (4)</h2>
    <!-- Sentence cards for level: "complex" -->
  </section>

</div>
</body>
</html>
```

### Sentence Card Template
```html
<div class="sentence-card">
  <div class="card-header">
    <span class="card-num">{id}.</span>
    <span class="badge badge-{level}">{level}</span>
  </div>
  <div class="sentence-french">{target}</div>
  <div class="sentence-english">{translation}</div>
  <div class="sentence-note">{grammar_note}</div>
</div>
```

## CSS Class Mapping

Use these classes from `styles.css`:
- `.container`: outer wrapper (max-width: 800px)
- `header`: title section (blue background)
- `section`: grouping by difficulty
- `section > h2`: section heading with badge
- `.section-badge.badge-sentences`: badge for "Sentences" sections
- `.sentence-card`: white card with border
- `.card-header`: header of card (ID + badge)
- `.card-num`: sentence number (bold)
- `.sentence-french`: the French sentence (bold, larger text)
- `.sentence-english`: the English translation (smaller, gray)
- `.sentence-note`: grammar note (italic, blue)
- `footer`: footer text
- Dark mode support via `@media (prefers-color-scheme: dark)`

## Field Mapping

From JSON to HTML:
| JSON Field | HTML Element | CSS Class |
|---|---|---|
| `word` | `<h1>` | header |
| `base_form` | `<p class="focus">` | header |
| `word_type` | `<p class="focus">` | header |
| `generated_at` | `<p class="meta">` | header |
| `sentences[i].id` | `<span class="card-num">` | card-num |
| `sentences[i].level` | badge class (badge-simple, badge-medium, badge-complex) | section grouping |
| `sentences[i].target` | `<div class="sentence-french">` | sentence-french |
| `sentences[i].translation` | `<div class="sentence-english">` | sentence-english |
| `sentences[i].grammar_note` | `<div class="sentence-note">` | sentence-note |

## Error Handling

- **Missing JSON file**: If a JSON file is malformed, skip it and report: "Skipped [file]: invalid JSON"
- **Missing field in JSON**: If a sentence is missing `target`, `translation`, or `grammar_note`, use placeholder or skip
- **No unrendered files**: Display: "All sentence files are already rendered. No new HTML files created."

---

## Dark Mode
The stylesheet (`styles.css`) includes dark mode support via `@media (prefers-color-scheme: dark)`. No special action needed; HTML automatically adapts to user's OS preference.
