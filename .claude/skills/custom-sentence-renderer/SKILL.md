# custom-sentence-renderer Skill for French Learning Assistant

## Overview
Render custom sentence pairs from `data/custom-sentences/` JSON files into interactive HTML pages. Each sentence displays English by default with a toggle button to reveal the French translation. Vocabulary words are displayed in a collapsible reference section.

## Invocation
`render custom sentences`

## Context from CLAUDE.md

- **Target language**: French (A1)
- **Instruction language**: English
- **Learner profile**: Native Slovak speaker, prefers strict corrections and interactive, toggle-based learning

---

## Workflow

### Step 1: Discover JSON Files
- Scan `data/custom-sentences/` for all files matching `custom-sentences-v*.json`
- Sort by version number (v1, v2, v3, etc.)
- Track which versions have already been rendered (check for corresponding `.html` files in `data/custom-sentences-html/`)

### Step 2: For Each JSON File
- Read the file
- Extract sentences array and words array
- Identify the version number from filename (e.g., `custom-sentences-v1.json` → version 1)

### Step 3: Generate HTML
**Page Structure:**
- `<html lang="fr">` (French language tag)
- Header: Version, creation date, sentence/word counts
- **Sentences Section**:
  - Each sentence displayed as a card
  - English visible by default
  - Toggle button to show/hide French translation
  - Display focus words prominently (with `*` markers from JSON)
  - Show focus words in a note below each pair if applicable
- **Vocabulary Section** (collapsible):
  - Table of all words with English/French translations
  - Show gender, alternatives, word type
  - Collapse/expand button
- Footer: "Generated by French Learning Assistant · Custom Sentences · [version]"

### Step 4: Interactive Features
- **Toggle Button**: Click icon/button next to each sentence to reveal/hide French
- **Vocabulary Collapse**: Click to expand/collapse the vocabulary reference
- **Focus Word Highlighting**: Preserve `*focus*` markers visually (bold or highlighted)
- **Dark/Light Mode**: Use shared `../../styles.css` (light + dark mode support)

### Step 5: Save HTML Files
- Create `data/custom-sentences-html/` directory if it doesn't exist
- Save each rendered version as `custom-sentences-v[N].html`
  - Example: `custom-sentences-v1.json` → `custom-sentences-v1.html`
- Use the same stem name for easy tracking

### Step 6: Update index.html
- Add a new section for "Custom Sentences" if it doesn't exist
- For each rendered version, add a link to the HTML file
- Show version number and creation date
- Update sentence/word counts
- Example: `Custom Sentences v1 (8 sentences, 12 words) — March 17, 2026`

### Step 7: Output Summary
Display a summary showing:
- Number of JSON files found and rendered
- List of generated HTML files
- Links to view each rendered version
- Total sentences and words registered

```html
    </div>

  </section>

<style>

  .vocab-table th { background: var(--bg-secondary); font-weight: bold; }
</style>

<script>
  document.querySelectorAll('.toggle-btn').forEach(btn => {
    btn.addEventListener('click', function() {
      const french = this.nextElementSibling;
      french.classList.toggle('hidden');
      this.textContent = french.classList.contains('hidden') ? '🔄 Reveal' : '🔄 Hide';
    });
  });

  document.querySelector('.collapse-btn')?.addEventListener('click', function() {
    const table = document.getElementById('vocabTable');
    table.classList.toggle('hidden');
    this.textContent = table.classList.contains('hidden') ? '▶ Show' : '▼ Hide';
  });
</script>
</body>
</html>
```

## Quality Checklist

Before outputting HTML, verify:
- ✓ All JSON files in `data/custom-sentences/` are discovered
- ✓ English sentences display clearly by default
- ✓ Toggle buttons work and reveal/hide French translations
- ✓ Focus words are highlighted with `*...*` formatting
- ✓ Vocabulary table shows all words with correct gender/type
- ✓ HTML uses shared stylesheet (light/dark mode compatible)
- ✓ Links in `index.html` are relative paths (work from any location)
- ✓ All HTML is valid and semantic
- ✓ Interactive features have accessible button labels

---

## Error Handling

- **No JSON files found**: "No custom sentence files found in `data/custom-sentences/`. Use `register-sentence` to create some."
- **Missing `data/custom-sentences-html/` directory**: Create it automatically
- **Malformed JSON**: Skip the file and report the error, continue with others
- **No index.html**: Create a warning that manual update may be needed
